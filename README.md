# 🛢️ India Energy Supply Chain — Risk Dashboard

AI-powered geopolitical risk monitoring & scenario analysis platform built for ET AI Hackathon 2.0.

## 🔴 Live Demo
[View Live Dashboard](https://et-hackathon-energy-risk-d9fjggtltsl63zesf7l2wt.streamlit.app/)

## 🎯 Problem Statement
India imports 88% of its crude oil, with 40-45% transiting through the Strait of Hormuz. Traditional supply chain tools cannot model geopolitical scenario impacts in real time. This platform solves that.

## ✨ Features
- **Live Risk Feed** — Monitors geopolitical headlines and scores each one 0-10 for supply chain risk
- **Scenario Modeller** — Simulates 3 major disruption scenarios with cascading economic impacts
- **Procurement Recommendations** — Generates actionable rerouting options for each scenario

## 🛠️ Tech Stack
- Python 3.13
- Streamlit
- Gemini API
- feedparser

## 🚀 Run Locally
pip install streamlit feedparser python-dotenv
python -m streamlit run dashboard.py
